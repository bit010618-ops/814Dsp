# Module and data map

- `full/source/`：章节、逐页转写、真题与组件内容模型。
- `full/tools/`：内容模型、PDF 生成和审计工具。
- `full/tests/`：内容模型与生成约束验证。
- `full/outputs/`：可复建的组件或最终 PDF 输出。
- `sample/`：已验证的版式样稿基线；`legacy_full/`：仅作可复用绘制资产基线，非当前成品。

# Permanent product constraints

- 不遗漏原课件唯一信息，保持原章节、知识点和顺序。
- A4 纵版、彩色、传统学术教材风；正文自然续排。
- 展示公式统一 11 pt；行内公式 17.5 pt、相对正文基线下移 3 pt；正式数学排版和竖式分数不可例外。
- 图、题图去水印和课件模板时不得损失任何有效技术信息；不清晰或被遮挡时修复或重绘。
- 每章末有重点精练与补充真题，所有纳入真题在书末给出详细解答并回填页码。

# Known risks

- 第一章和全书尚未完成最终自然分页与覆盖验收；现有组件不能被误报为完整章节。
- 真题图重绘／修复和最终答案页码回填尚未完成。
- 当前工作区有未提交的内容模型、组件、诊断和临时文件；必须逐文件判断，禁止误提交。

# Latest decisions

1. 动画状态以技术信息完整且可读性最佳为准，不能机械使用最后一帧。
2. 图内文字、公式、刻度、图例、题注不得与线条、坐标轴或其他文字重合。
3. 课堂圈画仅在承担必要解释时保留；否则恢复为干净教材状态。
4. 课内练习和真题使用纯白题干与自然书写区，不用黄色底、边框、横线或“练习”标签。
5. 跨章节题：独立小题可分置；关联题整体放在所需知识最晚章节。
6. 附录保留 A、B、C、D、E、F、I；不单列 G、H。
7. 封面、目录和附录采用蓝金教材风；目录、答案索引均以最终分页回填。
8. 可读性优先，但不得以此省略唯一推导、单位、图形标注或例题步骤。
9. 本讲义仅覆盖 DSP 基础知识；所有 MATLAB 函数、绘图、程序代码、音频读写和实验内容均排除在成品之外。
