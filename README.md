# 814 DSP 讲义项目

本仓库用于制作《数字信号处理讲义》：将 MOOC 课件重制为适合彩色 A4 打印、装订与考研复习的传统教材风讲义。

- `docs/requirements/`：需求访谈确认、最终方案和变更记录。
- `HANDOFF.md`：兼容性的上下文恢复入口；完整历史已归档。
- `sample/`：代表性 5–10 页样稿及其可修改源文件（制作中）。
- `full/`：全文讲义的制作、审计与可修改源文件区域。

任何继续此项目的 Codex 会话，先读 `AGENTS.md`、`.codex/CONTEXT.md` 和 `.codex/TASK.md`；`HANDOFF.md` 仅作为兼容入口。
