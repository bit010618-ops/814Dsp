# 当前源课件的内容基线对比

审计日期：2026-07-22

## 方法与边界

以当前主课件 `MOOC课件-合并版.pdf` 的 1056 页文本层为真相来源，分别与两个
可打印候选 PDF 做逐页归一化文本匹配。此指标用于发现明显缺口和排序补写工作，
**不能**单独证明内容完整：图形、公式位置、颜色语义和经合理改写后的表述仍须逐页
视觉核验。

完整页级证据：

- `full/artifacts/legacy_current_coverage.json`
- `full/artifacts/reference24_current_coverage.json`

## 当前决策

1. `legacy_full/` 继续作为可修改的排版、公式渲染和图形重绘资产库；其 119 页 PDF
   不参与当前成品内容覆盖判定。
2. 用户提供的二十四稿作为本轮**内容补写优先基线**：其章节正文、真题导航和连续
   教材节奏可用于加速转写，但其中每一处图形、水印、公式、题目和源课件缺口仍必须
   以主课件为准复核和修正。
3. 当前 `full/` 的八章生成器必须对每个源页记录 `source_page`、内容转写状态、图形
   处置和渲染核验结果；43 个低覆盖页必须在章节验收前逐页关闭，其他页也不得因
   匹配率较高而跳过原图／公式检查。
