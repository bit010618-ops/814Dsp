# MOOC DSP A4 讲义代表性样稿设计

## 目标

从 MOOC 数字信号处理课件中选取一段具有代表性的内容，制作 5–10 页可打印的彩色 A4 纵版讲义样稿，用于锁定全书重制的内容合并规则、视觉风格、公式规范、图形处理和真题训练结构。

## 内容边界

- 课件是内容与顺序的唯一主线；动画递进页只保留最终完整状态，且最终内容必须覆盖每页新增信息。
- 正文、公式、图形、表格、标注、箭头与颜色强调不擅自删减；明确错误直接修正。
- 原有例题可在原解答步骤之间补足推导，形成完整连续解答。
- 真题仅作为每章末尾“高频考点 + 真题训练”的补充；样稿需展示至少一种真题训练版式，但最终选题需由章节关联和真题统计支持。

## 页面系统

- 成品：A4 纵版、彩色、传统教材风、适合单双面打印与装订。
- 页眉：左侧“数字信号处理讲义”，右侧当前章名；页脚仅居中页码。
- 目录：最终全书列章、主要节和小节；样稿无需生成完整目录，但应预留其视觉体系。
- 章节：章从新页开始；章内保持连续阅读和自然留白，避免空白批注栏和无意义大空白。
- 字号：正文、各级标题、图注、表格、公式独立固定层级；公式一般不随页面变化而缩放。

## 图形与图像规则

- 无水印且按实际打印大小清楚的原图可直接使用。
- 出现明显锯齿、模糊、糊字、细线断裂、水印或裁切不完整的图，改用可编辑重绘图。
- 重绘图必须保留原图拓扑、箭头、标签、刻度、比例和颜色语义。

## 样稿选段

扫描后在同一知识段内选择：

1. 至少一组动画递进页，能证明“最终完整状态合并”正确；
2. 至少一个正式数学公式和一段解释正文；
3. 至少一张图形或波形/框图；
4. 至少一个原课件例题或可关联的真题训练题。

若没有单一段落同时满足，样稿可以使用相邻段落，但仍按原课件顺序展示。

## 验收方式

- 每张样稿页标明对应的源 PDF 页范围或动画页组。
- PDF 以 300 dpi 渲染检查，无截断、重叠、乱码、模糊或明显锯齿。
- 抽检文字、公式、箭头、标注和颜色强调是否与源内容一致。
- 检查页眉页脚、页码、边距、留白和统一字号体系。

## 恢复与交付

- 扫描记录、页映射、生成源、候选 PDF 和渲染图保存在独立样稿工作目录。
- 每个已验证里程碑更新进度记录；样稿通过后将项目迁入用户 Git 仓库中的独立目录并提交推送。
